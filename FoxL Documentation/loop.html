Loops

In FoxL, a loop is a control construct that executes a block of code a number of times. There are two main types of loops in FoxL: for loops and while loops.

The for loop is used to repeat a block of code a predetermined number of times. The syntax of the for loop is as follows: 

                    for ({initialization variable}; {condition}; {jump}) {
                        {code block}
                    }

{initialization variable} is executed once before the loop starts. {condition} is checked after each loop. If the condition is true, the loop will continue. If the condition is false, the loop will terminate. {jump} is executed after each loop. It can be a number, an expression, or a command. For example, the following code will print the numbers 1 through 10: 

                    for (var i = 1; i <= 10; i++) {
                        print(i);
                    }

The while loop is used to repeat a block of code until a certain condition is satisfied. The syntax of the while loop is as follows: 

                    while ({condition}) {
                        {code block}
                    }

The loop will continue to execute the block of code until the condition is false.

For example, the following code will print even numbers from 1 to 20: 

                    i = 1;
                    while (i <= 20) {
                        if (i % 2 == 0) {
                            print(i);
                        }
                        i++;
                    }

Loops are a powerful tool that can be used to perform many types of tasks in FoxL.
